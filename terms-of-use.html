Al acceder a nuestro sitio web y utilizar los servicios proporcionados, usted acepta cumplir con estos Términos de Uso. Si no está de acuerdo con ellos, por favor, no utilice este sitio web ni sus servicios.

Usted se compromete a no realizar acciones que puedan dañar, sobrecargar o deteriorar la funcionalidad del sitio web, incluyendo, pero no limitado a, la distribución de virus o software malicioso.

Nos reservamos el derecho de suspender o terminar el acceso al sitio para los usuarios que violen estas restricciones o se comporten de manera poco ética o ilegal.

No somos responsables de ningún daño, pérdida o perjuicio que surja del uso de nuestro sitio web o su contenido. Esto incluye, pero no se limita a, daños directos, indirectos, incidentales o consecuentes.
